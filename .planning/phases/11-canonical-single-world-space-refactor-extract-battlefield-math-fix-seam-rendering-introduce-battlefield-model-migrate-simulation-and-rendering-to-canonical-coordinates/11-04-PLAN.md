Rewrite the rendering pipeline to draw canonical entities directly from Battlefield.entities into both viewports. Delete the crossed-entity remap paths (game_map_crossed_world_point, game_apply_crossed_direction, game_render_seam_rt) and the seamRT RenderTexture.       
       ```c
       // Draw all Battlefield entities visible in the current viewport.
       // Entities are in canonical world space; the active Camera2D handles
       // projection. No ownership branching, no remap. (per D-19)
       static void game_draw_canonical_entities(const Battlefield *bf) {
           for (int i = 0; i < bf->entityCount; i++) {
               const Entity *e = bf->entities[i];
               if (!e || !e->alive || !e->sprite) continue;
               entity_draw(e);
           }
       }
       ```
       - This function is intentionally simple: draw every entity. The Camera2D + scissor mode automatically clips entities outside the viewport. No visibility culling needed at 128 max entities.
